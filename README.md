Запустите PostgreSQL.

Создайте базу данных с любым названием

Откройте файл со скриптом базы данных (db.sql)

Выполните скрипт в PostgreSQL (через pgAdmin или psql)

Скачайте папку rabota и продолжайте работу в ней

Установите зависимости в вашей среде разработки pip install -r requirements.txt

Откройте main.py и проверьте строку подключения к базе данных 

В папке проекта выполните команду uvicorn main:app --reload

Далее откройте файл gui.ipynb и запустите его
